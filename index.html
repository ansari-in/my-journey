Core Chapters

Four major phases that transformed me from a beginner learner into a project-driven full-stack developer.

2020 • Beginning

Chapter 01: The first code lines

A TV ad triggered my curiosity, then I searched "website coding" and entered web development through HTML lessons. This phase built discipline: watch, practice, repeat.

2021 • Frontend grind

Chapter 02: Design through replication

I built Flipkart, YouTube, and Firefox-inspired interfaces plus custom sign pages. Copying real interfaces trained my eye for spacing, layout, and visual consistency.

2021 → 2022 • Backend

Chapter 03: From frontend to full-stack

I built Taqdis Computers with service listings, product management, login panel, and user metadata capture. It was manually coded and became my first project with practical real-world value.

2023 → 2025 • Scale up

Chapter 04: Production mindset

I shipped an I-Card generator, quiz portal, docs platform using Nextra, and API-based apps on Vercel/Render/Firebase, balancing performance with free-tier constraints.
